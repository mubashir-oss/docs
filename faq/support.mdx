## Getting Support

### Contact Channels

- **Email** - [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com)
- **Documentation** - Comprehensive guides at [docs.unsiloed.ai](/)
- **Dashboard** - Technical support through your account

## Technical Support

### Report a Bug

Email [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com) with:
- Issue description and steps to reproduce
- Error messages or screenshots
- API logs (if applicable)

### Request a Feature

Email [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com) with your suggestion, use case, and why it would be valuable.

### Integration Help

We provide:
- Detailed documentation and code examples
- Python and JavaScript SDKs with samples
- Technical consultation (Pro/Enterprise)
- Custom integration support (Enterprise)

## Account Support

### Update Payment Info

Dashboard → Settings → Billing → Update payment method

### Cancel Subscription

Dashboard → Settings → Billing → Cancel Subscription

Or email [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com)

### Password/Email Changes

Use "Forgot Password" on login page or email support for account email changes.

## Enterprise Support

Enterprise plans include:
- Dedicated support engineer
- Priority response (2-4 hours)
- 24/7 emergency support
- Custom SLAs and uptime guarantees
- Quarterly business reviews

[Contact us](mailto:hello@unsiloed-ai.com) to upgrade to Enterprise Support.
