## Share Feedback

We value your input! Share feedback through:

- **Email** - [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com)
- **Dashboard** - Use the feedback widget in your account
- **Scheduled calls** - [Sign up on Unsiloed AI](https://cal.com/aman-mishra-p0ry57/15min)

### What We're Looking For

- Product features and UI/UX improvements
- API and SDK feedback
- Documentation clarity
- Performance and reliability issues
- Use cases and workflows

All feedback helps shape our roadmap!

## Feature Requests

### Request a Feature

Email [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com) with:

1. Your use case and problem to solve
2. How the feature would benefit you
3. Examples or mockups (if applicable)

### How We Prioritize

Based on customer impact, use case value, strategic alignment, and technical feasibility. Enterprise customers get priority consideration.

### Notifications

We'll notify you when your requested feature is implemented and invite you to beta test if applicable.

## Bug Reports

Email [hello@unsiloed-ai.com](mailto:hello@unsiloed-ai.com) with:
- Clear description and reproduction steps
- Expected vs. actual behavior  
- Error messages, screenshots, or API logs

We'll acknowledge within 24 hours and keep you updated on resolution.
