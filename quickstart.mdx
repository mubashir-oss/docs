## Welcome to Unsiloed AI

Transform your unstructured financial documents into structured data with our powerful AI-driven platform. This guide will help you get started quickly with our document processing capabilities.

### Authentication & Setup

<AccordionGroup>
  <Accordion icon="key" title="Get your API key">
    1. Sign up for an account at [Unsiloed AI Dashboard](https://www.unsiloed-ai.com/dashboard)
    2. Navigate to the API Keys section
    3. Generate a new API key for your project
    4. Keep your API key secure - you'll need it for all API requests
    
    <Note>
      Your API key provides access to all Unsiloed AI services. Keep it confidential and never expose it in client-side code.
    </Note>
  </Accordion>
  
  <Accordion icon="code" title="Make your first API call">
    Test your setup with a simple document extraction request:
    
    ```bash
    curl -X POST "https://api.unsiloed-ai.com/v1/extract" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "document_url": "https://example.com/financial-report.pdf",
        "extraction_type": "financial_data"
      }'
    ```
    
    <Tip>
      Replace `YOUR_API_KEY` with your actual API key from the dashboard.
    </Tip>
  </Accordion>
</AccordionGroup>

### Core Document Processing

<AccordionGroup>

<Accordion icon="file-pdf" title="Process your first document">
  Upload and process a financial document using our extraction API:
  
  ```python
  import requests
  
  # Upload a document for processing
  url = "https://api.unsiloed-ai.com/v1/extract"
  headers = {
      "Authorization": "Bearer YOUR_API_KEY",
      "Content-Type": "application/json"
  }
  
  data = {
      "document_url": "https://example.com/annual-report.pdf",
      "extraction_type": "comprehensive",
      "output_format": "json"
  }
  
  response = requests.post(url, headers=headers, json=data)
  result = response.json()
  
  print(f"Job ID: {result['job_id']}")
  print(f"Status: {result['status']}")
  ```
</Accordion>

<Accordion icon="chart-line" title="Extract financial data">
  Use our specialized financial data extraction for structured output:
  
  ```javascript
  const extractFinancialData = async (documentUrl) => {
    const response = await fetch('https://api.unsiloed-ai.com/v1/extract', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        document_url: documentUrl,
        extraction_type: 'financial_metrics',
        include_tables: true,
        include_charts: true
      })
    });
    
    return await response.json();
  };
  ```
</Accordion>
